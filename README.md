# Customer-Funnel-Analysis-Project
This project was premised on anlysizing customer conversion funnel data to identify drop-off points. This is to help elicit actionable evidence and decisions to improve customer retention, address customer churn rate, by turning raw behavioral data into a strategy for growth 
Customer Funnel Analysis Using Python

Objective:
Analyze customer conversion funnel data to identify drop-off points.

Tools:
Python
Pandas
Matplotlib

Steps:
1 Import dataset
2 Clean data
3 Calculate conversion rates
4 Visualize funnel performance

Insights:
Identify stages where most customers exit the funnel.
